# Example feature brief

Title: Sign-up screen with email/password

Summary
- Add a React Native iPhone screen that lets users sign up with email and password.

Goals
- Allow new users to create accounts
- Validate email format and password strength

User story
- As a new user, I want to create an account with email and password so I can access the app.

Acceptance criteria
- Email and password fields present
- Client-side validation for email format and minimum password length
- Unit tests for validation logic

Tasks (sample)
- Task 1: Create SignUpScreen component (feature/pm-signup-screen)
- Task 2: Implement form validation and tests
- Task 3: Add navigation entry and basic styling
